# EventOS

Sistema web de gestión de entradas con Flask.
Incluye:
- Módulo de ventas
- Escaneo de QR
- Estadísticas y reportes
- Configuración de plantillas
## Estadísticas (GLOBAL)
La pantalla de **Estadísticas** muestra indicadores globales:
- Total de ventas (conteo de entradas vendidas con estado VIGENTE/UTILIZADA)
- Entradas utilizadas
- Ingresos totales y precio promedio
- Series de ventas de los últimos 30 días
- Top por tipo de entrada, por evento, y por vendedor (agregado global)

*Nota:* En esta versión no hay filtro por evento; todas las métricas son globales.
